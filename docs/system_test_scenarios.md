# UB Karaoke Системийн Тест Кейс болон Сценари (System Test Scenarios)

Энэхүү баримт бичиг нь системийн үндсэн функцуудыг бүрэн шалгах тест кейсүүдийг агуулсан болно.

---

## 1. Хэрэглэгчийн Бүртгэл болон Нэвтрэх (User Auth)

| ID | Сценари (Scenario) | Алхам (Step) | Хүлээгдэж буй үр дүн (Expected Result) |
|:---|:---|:---|:---|
| TC-01 | Шинэ хэрэглэгч бүртгүүлэх | Бүртгүүлэх хэсэгт нэр, имэйл, утас, нууц үг оруулах | Бүртгэл амжилттай болж, баталгаажуулах код илгээгдэнэ. |
| TC-02 | OTP-оор нэвтрэх | Утасны дугаараа оруулаад OTP код авах | Код утсанд ирж, амжилттай нэвтэрнэ. |
| TC-03 | Нууц үгээр нэвтрэх | Имэйл болон нууц үгээ оруулан нэвтрэх | Профайл хуудас руу шилжинэ. |
| TC-04 | Нууц үг шинэчлэх | "Нууц үг мартсан" холбоос дээр дарж имэйлээ оруулах | Нууц үг солих холбоос имэйлээр ирнэ. |

## 2. Захиалгын Урсгал (Booking Flow)

| ID | Сценари (Scenario) | Алхам (Step) | Хүлээгдэж буй үр дүн (Expected Result) |
|:---|:---|:---|:---|
| TC-05 | Салбар болон Өрөө сонгох | Салбарын жагсаалтаас сонгож, өрөөний мэдээлэл үзэх | Өрөөний үнэ, багтаамж, зураг харагдана. |

| TC-08 | Захиалга цуцлах | Pending эсвэл Reserved захиалгыг цуцлах | Төлөв **CANCELLED** болж, хэрэглэгчийн түүхэнд хадгалагдана. |
